# Transcar Landing Pages

Simple homepage that lists the landing pages being built for
Transcar Auto Shippers, Inc.

## Pages listed

1. Hawaii Car Shipping
2. Guam Car Shipping
3. Automobile Transport

Entries are display-only for now (no links yet).

## Files

- `index.html` — homepage markup
- `styles.css` — styling, following the [transcar.com](https://transcar.com/) design (navy / red / cyan palette, condensed uppercase headings)
- `assets/logo.webp` — Transcar logo

## Preview

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
